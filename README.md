# 🧹 Bilibili 抽奖动态清理工具

自动扫描当前 B 站账号的全部动态，识别已开奖的互动抽奖动态并批量删除。

## 快速开始

```bash
# 1. 安装依赖
pip install requests browser-cookie3 pycryptodomex pywin32

# 2. 运行
python delete.py
```

程序会**自动从浏览器读取 cookie**，无需手动导出或保存任何敏感信息。

## 使用流程

1. 确保已在本地浏览器登录 bilibili.com
2. 运行 `python delete.py`
3. 选择是否开启调试模式
4. 选 **试运行（1）** 仅检测，或 **正式运行（2）** 删除已开奖动态

## 支持的浏览器

| 浏览器 | 自动读取 | 备注 |
|---|---|---|
| **360 极速浏览器 X** | ✅ 自动 | 首次使用会自动检测 |
| **Chrome** | ✅ 自动 | 通过 `browser-cookie3` |
| **Edge** | ✅ 自动 | 通过 `browser-cookie3` |
| **Firefox** | ✅ 自动 | 通过 `browser-cookie3` |

## 功能特性

- **🔒 零落盘** — cookie 从浏览器 → 内存 → 用完即弃，不写入任何文件
- **🎯 精准识别** — 4 层检测策略（关键词递归 + 字段匹配 + 正则 + API 验证）
- **⏳ 智能跳过** — 自动区分已开奖（可删）和未开奖（保留）
- **⏸️ 试运行模式** — 先看结果再决定是否删除
- **🕐 随机延迟** — 内置 2-4 秒间隔，降低限流风险

## 工作原理

```
浏览器 cookie → 内存暂存 → 拉取动态列表
                              ↓
                    检测是否为抽奖动态
                     ├─ 关键词递归搜索
                     ├─ 模块字段匹配
                     ├─ 文案正则匹配
                     └─ API 接口验证
                              ↓
                    检查开奖状态 → 未开奖 ⏭
                        ↓ 已开奖
                    执行删除（POST JSON）
```

## 注意事项

- 360 极速浏览器 X 有**常驻后台进程**，脚本会自动通过 PowerShell 绕过文件锁
- B 站 API 有频率限制，脚本已内置随机延迟
- 非互动抽奖（如转发抽奖）可能无法完全识别
- 如浏览器 cookie 过期，请重新登录 bilibili.com 后重试
